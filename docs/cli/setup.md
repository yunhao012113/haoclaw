# `haoclaw setup`

Initialize `~/.haoclaw/haoclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
haoclaw setup
haoclaw setup --workspace ~/.haoclaw/workspace
```

To run the wizard via setup:

```bash
haoclaw setup --wizard
```
